# COMP3011 Search Engine Tool

## Overview
This is a Python-based command-line search engine tool built for COMP3011 Coursework 2. It crawls `https://quotes.toscrape.com/`, builds an inverted index of word frequencies, respects a 6-second politeness window, and allows users to search for single or multi-word phrases.

## Installation & Setup
1. Ensure you have Python 3 installed.
2. Clone this repository.
